# Legislative Tracking System for Dagupan City Government

A comprehensive digital solution for modernizing legislative processes and enhancing transparency in the Sangguniang Panlungsod Office.

## 🏛️ Project Overview

This system replaces manual, paper-based workflows with an integrated digital platform, providing:

- **Web-Based Legislative Portal** - Public access to legislation, agendas, and tracking
- **E-Session System** - Paperless council sessions with electronic signatures
- **Document Management** - Comprehensive tracking of ordinances, resolutions, and communications

## 🚀 Key Features

### Public Portal
- Real-time access to approved legislation and agendas
- Document tracking and search functionality
- Statistical reports and analytics
- Public accreditation and registration
- Committee member information

### E-Session System
- Digital agenda management
- Real-time electronic signatures
- Paperless session environment
- Live updates and transcriptions

## 🛠️ Technology Stack

- **Frontend**: React.js with TypeScript
- **Backend**: Node.js with Express
- **Database**: PostgreSQL
- **Real-time**: Socket.io
- **PDF Processing**: PDF.js, jsPDF
- **Authentication**: JWT
- **UI Framework**: Material-UI

## 📁 Project Structure

```
├── frontend/          # React web application
├── backend/           # Node.js API server
├── database/          # Database schemas and migrations
├── docs/             # Documentation and API specs
├── docker/            # Docker configuration
└── scripts/           # Deployment and setup scripts
```

## 🚀 Getting Started

1. **Clone the repository**
2. **Install dependencies**: `npm install` in both frontend and backend directories
3. **Set up database**: Configure PostgreSQL connection
4. **Start development servers**: `npm run dev`

## 📋 Requirements

- Node.js 18+
- PostgreSQL 14+
- Modern web browser
- Tablet/laptop for e-Session functionality

## 🔐 Security Features

- Role-based access control
- JWT authentication
- Secure document handling
- Audit logging
- Watermark protection for sensitive documents

## 📱 E-Session Requirements

- Internet-connected tablets/laptops
- Stylus support for electronic signatures
- Real-time synchronization
- Offline capability for critical functions
